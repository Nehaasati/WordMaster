рҹҺ® WORD MASTER   is a real-time multiplayer word game where players compete to find words across categories using a shared set of letters.



рҹҡҖ Features
рҹҺ® Multiplayer lobby system 
рҹ”Ө Random letter generation
вң… Word validation (dictionary + category + letters)
рҹ§® Scoring system with bonuses
рҹӣ’ In-game shop (letters & power-ups)
рҹҺӯ Characters with abilities
вҡЎ Real-time gameplay using SignalR


рҹ•№пёҸ How the Game Works
Host Creates Lobby
A player (host) creates a new lobby.
The system generates a lobby ID / invite code.

Choose Character
The host selects one of four characters:
рҹҰү Ugglan вҖ“ +3 bonus points for words longer than 5 letters
рҹҗҶ Leopard вҖ“ +3 bonus points for words submitted within 60 seconds
рҹҗӯ Musen вҖ“ +1 bonus point for words shorter than 4 letters
рҹҗ» BjГ¶rnen вҖ“ Immune to the Freeze chaos event

Second Player Joins
The second player joins using:
Invite link or
Lobby code
The player also selects a character.
Game Start
The host starts the game when both players are ready.

рҹ”Ө Gameplay
Players receive a shared set of random letters.
The goal is to:
Find valid words
Match categories
Submit as many correct words as possible

рҹӣ’ Shop & Power-ups
During the game, players can use points in the shop:
рҹ”Ө Buy extra letters (including Swedish vowels like Г…, Г„, Г–)
вқ„пёҸ Freeze вҖ“ temporarily blocks the opponent
рҹҺҙ Joker Card
Gives a random letter
If used in a word вҶ’ double points

вҡ”пёҸ Player Interaction
Players can:
Attack each other using power-ups (e.g. Freeze)
Gain advantages through shop items
Strategically use Joker cards

рҹҸҒ Game End
The game ends when:
A player finishes all categories or
The round is completed
Final scores are calculated:
Base points
Bonus points (character abilities)
Shop effects
